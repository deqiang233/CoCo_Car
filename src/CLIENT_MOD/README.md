# CLIENT_MOD - DayZ模组集合

这个仓库包含了DayZ服务器的客户端模组文件。

## 模组列表

### CoCo_Car
- **作者**: deqiang233
- **功能**: DayZ载具管理系统
- **特性**:
  - 载具自动绑定系统
  - 支持多种载具类型
  - 管理界面（分号键打开）
  - 与VehicleSystem_V2集成

### VPPAdminTools
- **功能**: 管理员工具包
- **包含**: 完整的管理员管理界面和工具

### VPPStaticItems
- **功能**: 静态物品管理

### ZomBerry
- **功能**: 管理员工具

### assets
- **功能**: 资源文件

## 安装说明

1. 将整个`CLIENT_MOD`文件夹复制到你的DayZ服务器目录
2. 确保服务器配置正确加载这些模组
3. 重启服务器

## 使用说明

### CoCo_Car载具管理系统
1. 启动载具自动绑定到玩家账户
2. 按分号键(;)打开管理界面
3. 查看已绑定的载具信息
